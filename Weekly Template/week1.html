Section 3: Basics of C#

C# (pronounced C-sharp) is a versatile and modern programming language developed by Microsoft. Released in 2000, C# is designed for building robust and scalable applications on the Microsoft .NET framework. It combines the power of C++ with the simplicity of Java, making it a popular choice for a wide range of software development tasks.

C# supports object-oriented programming principles, providing features such as encapsulation, inheritance, and polymorphism. It boasts a strong type system that enhances code reliability and readability. The language is statically typed, meaning variable types must be declared at compile-time, aiding in early error detection.

C# excels in creating Windows desktop applications, web applications using ASP.NET, and game development through the Unity game engine. It also supports asynchronous programming with features like async/await, facilitating the development of responsive and efficient applications.

With a rich standard library and continuous updates, C# remains a preferred language for developers, enabling them to create high-performance and secure applications across various domains, from enterprise software to mobile and cloud-based solutions.

An example of C# code


using System;

class BasicCSharpExample
{
    // Entry point of the program
    static void Main()
    {
        // Variables and Data Types
        int age = 25;
        double height = 5.9;
        string name = "John";
        char gender = 'M';
        bool isStudent = true;

        // Output
        Console.WriteLine($"Name: {name}, Age: {age}, Height: {height}, Gender: {gender}, Is Student: {isStudent}");

        // Control Structures
        if (age >= 18)
        {
            Console.WriteLine("Person is an adult.");
        }
        else
        {
            Console.WriteLine("Person is a minor.");
        }

        // Looping
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine($"Iteration {i}");
        }

        // Arrays
        int[] numbers = { 1, 2, 3, 4, 5 };
        foreach (int number in numbers)
        {
            Console.Write(number + " ");
        }
        Console.WriteLine();

        // Functions
        int sum = Add(10, 20);
        Console.WriteLine($"Sum: {sum}");

        // Object-Oriented Programming
        Person person = new Person("Alice", 30);
        person.DisplayInfo();

        // Console.ReadLine() to keep the console window open
        Console.ReadLine();
    }

    // Function
    static int Add(int a, int b)
    {
        return a + b;
    }

    // Class for Object-Oriented Programming
    class Person
    {
        // Properties
        public string Name { get; set; }
        public int Age { get; set; }

        // Constructor
        public Person(string name, int age)
        {
            Name = name;
            Age = age;
        }

        // Method
        public void DisplayInfo()
        {
            Console.WriteLine($"Person: {Name}, Age: {Age}");
        }
    }
}
